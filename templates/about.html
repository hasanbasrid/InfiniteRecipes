What is Infinite Recipes?

Infinite Recipes is a web app with a social media approach about food combined with grocery shopping. It allows you to post recipes, comment on and rate recipes, and buy ingredients by getting redirected to a shopping website, as well as locating nearby grocery stores. Infinite Recipes is a school project for CS 490 (Design in Software Engineering).
Project completed in December 2020

Instructions and Ingridients

(How and with what technologies)

Ingridients:

  • Python
  • Flask
  • SocketIO
  • PostgreSQL
  • React JS
  • Spoonacular API
  • Google Maps API
  • HTML
  • CSS
  • Heroku

Instructions:

  • Start with Python as back-end language for databases and server-to-client communication
  • Add Flask (within Python) to help render the webapp
  • Add SocketIO (within Python) to allow server-client communication
  • Use PostgreSQL with Python to create tables and database to store webapp information
  • Use Spoonacular API to initially populate the database
  • Add (a lot of) React to build interactive user interfaces
    • Implement Google Oauth using React (react-google-login)
    • Use one of React's styling components: https://react.semantic-ui.com/
  • Use Google Maps API to show local grocery stores
  • Add a dash of HTML
  • Sprinkle a touch of CSS for styling
  • Plate (Deploy) the webapp via heroku
  • Enjoy
